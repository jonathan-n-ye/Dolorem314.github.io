Arrays

Arrays

Arrays in Java are a data structure that stores elements of the same type. Arrays are indexed and the first index is 0 and so on. The last element is index length - 1.

ArrayIndexOutOfBoundsException happens when you try to access a negative index or an index that is greater than or equal to the length of the array.

An array is an Object and it’s declared and initialized using:

//initialization
type[]... arrayName; //... represents any number of square brackets to determine the dimension of the array

//declaration
arrayName = new type[count]...; //declares an empty count * count2... array
//or
arrayName = {a, b, c, d} where a, b, c, d are values of datatype type

When declaring empty arrays:

  • numbers will be set to 0 or 0.0 depending on if the datatype is a floating point number
  • booleans will be set to false
  • objects will be set to null
  • char will be set to '\u0000’

Import array functions to know:

  • arrayName.length - returns the length of the array
  • arrayName[index][index2]… - returns the element at index/row index1, index/column index2…
  • Arrays.sort(arrayName) - sorts the array

For each loop

A For Each loop is a special kind of loop that traverses arrays and lists.

Format:

String[] arr = {"hi", "I", "like", "sharks"};
for(String a : arr){
	System.out.println(a);
}

2D Arrays

A 2D arrays is essentially an array within an array. Elements are found using 2darrayname[rowindex][columnindex]. Each inner array can have a different number of elements. This is called a ragged array.

Parallel Arrays

Parallel arrays are used when you try to have an array of objects that correspond to another array of objects of another datatype.

Passing Arrays

Arrays are passed by reference. Thus an edit to the array inside the method will edit the original array. This is called Persistent destruction of permanent data.

Sorting

Bubble sort: Swapping consecutive elements until all values are in order.

Insertion sort: Adding the first value of the old array to the correct position in the new array each time.

Selection sort: Adding the lowest value of an old array to the new array each time.

Searching

Sequential search: running through each index until you find the value.

Binary Search: Binary Search cuts the Array in half, checks the middle value and then cuts one of the sections in half again. It does it until it finds the value.
